ffmpeg命令的简单用法

-Map

基本语法

  • 输入文件索引：从 0 开始计数（如第一个输入文件为 0，第二个为 1）。
  • 流索引：从 0 开始计数（如第一个视频流为 0:v:0，第一个音频流为 0:a:0）。
  • 流类型指定符：
    • v：视频流（video）
    • a：音频流（audio）
    • s：字幕流（subtitle）
    • d：数据流（data）
    • t：时间戳流（attachment）

常见使用场景

1提取特定流

从输入文件中提取特定的视频或音频流。

  • 提取音频流：

    • 0:a:0：表示第一个输入文件（0）的第一个音频流（a:0）。

  • 提取视频流：

    • 0:v:0：表示第一个输入文件的第一个视频流。

合并多个输入流

将多个输入文件中的流合并到一个输出文件中。

  • 合并视频和音频：

    • 0:v：选择第一个输入文件的所有视频流。
    • 1:a：选择第二个输入文件的所有音频流。

  • 合并多个视频流：

    • 输出文件将包含两个视频流（分别来自 input1.mp4 和 input2.mp4）。

排除特定流

使用负号 - 排除不需要的流。

  • 排除第一个音频流：
    • 0：选择第一个输入文件的所有流。
    • -0:a:0：排除第一个音频流。

复杂映射（多输入多输出）

从多个输入文件中选择不同流并组合到输出文件中。

示例详解

提取所有音频流

  • -map 0:a：选择所有音频流。
  • -c:a copy：直接复制音频流（不重新编码）。

选择特定视频流并添加水印

  • -map 0:v：选择第一个输入文件的视频流。
  • overlay=10:10：将水印叠加到视频左上角。

合并多个音轨

  • 输出文件将包含视频流、第一个音频流和第二个音频流。

  1. 流索引从 0 开始：

    • 使用 ffprobe -i input.mp4 查看输入文件的流信息。

  2. 默认行为：

    • 不使用 -map 时，FFmpeg 会自动选择默认的视频流、音频流和字幕流。

  3. 流类型匹配：

    • 确保映射的流类型与输出格式兼容（例如，某些格式不支持多音轨）。

  4. 重新编码 vs 直接复制：

    • 使用 -c copy 可以直接复制流（无需重新编码），但需确保编码器兼容。
    • 如果需要调整参数（如分辨率、比特率），需使用对应的编码器（如 -c:v libx264）。

总结

  • -map 的核心作用：精确控制输入流到输出文件的映射，适用于复杂场景（如多音轨、多视频流合并、排除特定流等）。
  • 实际应用：结合 -c、-vf、-af 等参数，实现流的重新编码、过滤和处理。
  • 调试工具：使用 ffprobe 查看流信息，确保映射正确。

-c

ffmpeg -c（或 --codec）是 FFmpeg 中非常重要的一个参数，用于指定编码器/解码器，控制输入流或输出流的编解码方式。

基本语法

  • <stream_specifier>：可选，表示作用于哪种类型的流（如视频、音频、字幕等）
  • <codec_name>：具体的编码器或解码器名称（如 libx264, aac, copy 等）

常用用法详解

  1. 全局设置编码器（不推荐）

⚠️ 这会尝试对所有流使用 libx264 编码器，可能导致错误。通常应结合流类型使用。

  1. 指定特定流的编解码器（推荐）

视频流：-c:v 或 -vcodec

将视频流重新编码为 H.264 格式。

音频流：-c:a 或 -acodec

将音频流重新编码为 AAC 格式。

字幕流：-c:s 或 -scodec

将字幕流编码为 mov_text 格式（适用于 MP4 容器）。

六、注意事项

  • **不要滥用 -c copy**：确保容器格式支持你复制的流格式。例如 .mp4 不支持某些字幕格式。
  • 流索引问题：有时需要配合 -map 使用来精确选择哪些流参与处理。
  • 兼容性：不同平台对某些编码器的支持程度不同，尤其是硬件加速编码器。

-vf（简单滤镜图）和（复杂滤镜图）

ffmpeg -vf 参数用于指定应用于视频流的滤镜（filter）。FFmpeg 的滤镜系统非常强大，支持多种视频处理操作，如裁剪、缩放、旋转、添加水印等。以下是关于 -vf 参数的基本用法和一些常见示例。

基本语法

在这个命令中，-vf 后面跟着的是一个或多个滤镜及其参数，它们之间可以用逗号分隔来链式应用多个滤镜。

常见滤镜及使用示例

scale

调整视频分辨率。

将视频大小调整为 640x480。也可以仅指定宽度或高度，FFmpeg 会自动计算另一个维度以保持宽高比。

crop

裁剪视频。

从左上角开始裁剪出一个 640x360 的区域。

rotate

旋转视频。

将视频顺时针旋转 45 度（PI/4 弧度）。

transpose

旋转视频 90 度并可选地翻转。

transpose=1 表示顺时针旋转 90 度。

overlay

叠加图像（例如添加水印）。

在主视频的左上角(10,10)位置叠加水印图片。

drawtext

在视频上绘制文本。

在视频上添加白色的文字“Hello World”。

fade

添加淡入淡出效果。

在视频开头 3 秒内淡入，在第 30 秒开始 3 秒内淡出。

concat

连接多个视频片段（需要相同编码格式）。

虽然这不是直接通过 -vf 实现的，但对于简单的拼接任务，可以考虑使用 filter_complex 结合其他视频滤镜一起使用。

trim 和 setpts

修剪视频长度，并重置时间戳。

只保留从第 10 秒到第 20 秒之间的视频内容。
